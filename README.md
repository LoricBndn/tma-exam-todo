# Prérequis

- Avoir [Deno](https://deno.com) installé.

## Lancer l'application

Déno est sécurisé par défaut, il faut autoriser la lecture/écriture :

```bash
deno run --allow-read --allow-write src/mod.ts
```

## Lancer les tests

```bash
deno test --allow-read --allow-write
```

## Formater le code

```bash
deno fmt
```
